5 HTML Tags

head
This tag contains meta-information about the HTML document, such as its title, style sheet link and script.
body
This tag contains the visible content of the HTML document, including text, images, links, and other elements displayed in the browser.
p
This tag defines a paragraph of text.
a
This tag defines a hyperlink, used to link to other web pages or resources.
img
This tag indicates embedding the image into the document.
